# Examples

Reserved for small, executable usage examples once modules exist (AI
Coding Standards, section 21: "Exemplos de uso deverão permanecer
atualizados e executáveis"). `snkb --help`, `snkb version` and the
"pending" commands already work (see the root README's "Usage"
section); `snkb record` runs but `bootstrap.create_controller()`
raises `NotImplementedError` until Chapters 3-10 of the Module
Specifications are implemented.
